Уведомление безопасности

Система зафиксировала передачу активной сессии вашей учётной записи на стороннее устройство. Если вход выполняли не вы — немедленно свяжитесь со специалистом горячей линии для предотвращения несакцеированых действий.

  • Тип доступа: пересылка авторизационного токена / QR.
  • Источник: нераспознанное устройство / IP-адрес вне вашей геолокации.
  • Статус: ограниченный доступ до завершения проверки.
Важно: оператор не запрашивает пароли, коды из SMS, CVV и иные платежные данные.

Как действовать

  1. Выберите один из номеров горячей линии, указанных ниже.
  2. Сообщите оператору ваши ФИО и номер телефона для подтверждения личности.
  3. Опишите обстоятельства, при которых произошёл несанкционированный вход (например: получение подозрительной ссылки, передача данных третьим лицам или иная ситуация).
  4. Следуйте инструкциям оператора для восстановления и защиты учётной записи.
Частые вопросы

Почему сработала защита?
Замечен вход с нового устройства или передача сессии через QR/токен.

Что будет, если не подтвердить?
Часть функций кабинета будет временно недоступна до завершения проверки.

Нужно ли сообщать коды из SMS?
Нет, секретные коды и платежные данные не требуются.
